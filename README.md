# 이강림 / FrogRim

실시간 AI, 로보틱스, 그래픽스/엔진 시스템을 구현하고 성능으로 검증하는 개발자입니다.

> GitHub 프로필은 대표 프로젝트 6개로 신뢰를 만들고, 각 저장소 README는 문제, 역할, 스택, 실행법, 검증 결과를 보여주는 구조로 운영합니다.

## Focus

- WebRTC와 OpenAI Realtime API 기반 실시간 AI 제품
- LLM-first 로봇 제어와 로봇 데이터 수집/평가 파이프라인
- C++ 기반 그래픽스 알고리즘과 Unreal Engine 플러그인
- 접근성 문제를 다루는 모바일 AI 인터페이스
